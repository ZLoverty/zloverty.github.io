Introduction

corrLib (appears as corrLib.py) is a function library, a collection of functions, written in python which does various jobs in the research project on giant number fluctuations in bacterial suspensions. Typical jobs include image preprocessing, data extraction and data analysis.

What you should already know

This document assumes you have the following basic background:

  • General knowledge about python programming language
  • Basic understanding of statistical properties such as spatial and temporal correlation functions
  • Basics of image processing, such as smoothing and scaling.

Details

Image preprocessing

readseq(folder)
Parameters

folder: the full directory of folder from which image sequence of *.tif files are read.

Returns

fileList: a DataFrame containing columns (Name, Dir), where Name is the file name of each *.tif image without '.tif', and Dir is the full path of each corresponding image.

readdata(folder, ext='csv')

Parameters

folder: the full directory of folder from which data files are read.

ext: (optional) the extension of data files you are looking for.

Returns

fileList: a DataFrame containing columns (Name, Dir), where Name is the file name of each data file without extension, and Dir is the full path of each corresponding data file.

Data extraction

corrS(X, Y, U, V)

Parameters

X, Y: coordinates

U, V: velocity in x and y directions

Note - all the variables need to be in the shape similar to the raw images.

Returns

X, Y: coordinates, measure the distance between two points

CA: orientation correlation function

CV: velocity correlation function

corrI(X, Y, I)

Parameters

X, Y: coordinates

I: pixel intensity

Note - all the variables need to be in the shape similar to the raw images.

Returns

X, Y: coordinates, measure the distance between two points

C: pixel intensity correlation function

Data analysis

distance_corr(X, Y, C)

Parameters

X, Y: 2-D coordinates

C: correlation function

Note -- X, Y and C should be of the same shape

Returns

r_corr: DataFrame containing 2 columns (R, C). R is the 1-D distance and C is the value of correlation function at corresponding coordinates.
